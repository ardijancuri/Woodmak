# Woodmak

Custom WordPress/WooCommerce codebase for the Woodmak B2C + B2B furniture platform.

## Repository Scope

This repository intentionally includes only:

- Custom theme: `app/public/wp-content/themes/woodmak-store`
- Custom plugin: `app/public/wp-content/plugins/woodmak-b2b-core`
- Project docs: `docs/`

WordPress core, default themes, and third-party plugins are intentionally excluded.

## Requirements

- WordPress (installed separately)
- WooCommerce
- Polylang
- WooCommerce PDF Invoices & Packing Slips

## Install

1. Install WordPress on your environment.
2. Copy `app/public/wp-content/themes/woodmak-store` into your WordPress `wp-content/themes/`.
3. Copy `app/public/wp-content/plugins/woodmak-b2b-core` into your WordPress `wp-content/plugins/`.
4. Install/activate required third-party plugins.
5. Activate `Woodmak Store` theme and `Woodmak B2B Core` plugin.
